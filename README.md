## Milestone Description

This milestone focuses on **researching and reproducing code clone detection tools**.  

For each assigned tool, we have identified:
- The corresponding research paper  
- The associated GitHub repository (if available)  

In cases where no official repository or archived material was provided in the original paper, we have identified potential equivalent implementations using AI-assisted research. These alternatives will be evaluated and updated after testing to confirm whether they accurately reproduce the original tool’s functionality.
